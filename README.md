# network-scanner
My second cybersecurity project using Python 

## What this project does 
This program scans a target IP address and checks whether selected ports are open or closed.

## Features
- Takes an IP address aas input.
- Scans common ports.
- Displays open and closed ports.
- Uses Python socket programming.

## Technologies Used 
-Python 3.13
-Socket Library

## How to Run
python network_scanner.py
